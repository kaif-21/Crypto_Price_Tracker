# 🚀 Crypto Analytics Dashboard

A web-based **Crypto Analytics Dashboard** built using Python and Streamlit that allows users to track live cryptocurrency prices, analyze market trends, and calculate portfolio value in real-time.

---

## 📊 Features

### 🔹 Live Crypto Price Tracker

* Track real-time prices of cryptocurrencies
* Select coins like Bitcoin, Ethereum, and Dogecoin
* Displays current price in USD
* 7-day price trend visualization

### 🔹 Market Analytics Dashboard

* Displays market data for cryptocurrencies
* Shows top market trends
* Interactive charts using Plotly

### 🔹 Portfolio Tracker

* Enter the amount of cryptocurrency you hold
* Automatically calculates the portfolio value
* Helps estimate current investment worth

### 🔹 Interactive Data Visualization

* Graphs created using Plotly
* Real-time API data integration
* Clean and interactive UI with Streamlit

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Pandas**
* **Plotly**
* **Requests API**
* Data Source: CoinGecko API

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/crypto-analytics-dashboard.git
```

### 2️⃣ Navigate to the project folder

```bash
cd crypto-analytics-dashboard
```

### 3️⃣ Install dependencies

```bash
pip install streamlit pandas plotly requests
```

### 4️⃣ Run the application

```bash
streamlit run app.py
```
